Installation guide

Requirements
php ^8.0.2, composer

Run the following command

i)git clone git@github.com:parimal00/EventApp.git (inside xampp/htdocs)

ii) cd EventApp

iIi)composer install

iv)cp .env.example .env

v)php artisan key:generate

vi)npm install

vii)npm run dev

viii)php artisan serve
